## 安装说明 ##
**下载tar.xz文件.** 准备一个大于1G的空白U盘, 例如/dev/sdb
**U盘格式化为ext4分区.
```
    mkfs.ext4 -v /dev/sdb1
```** 解压下载文件
```
    mount /dev/sdb1 /mnt/usb
    tar xvapf 文件路径.tar.xz -C /mnt/usb
```
**安装GRUB-2.00到U盘MBR.
```
    grub-install --root-directory=/mnt/usb /dev/sdb
```**

**用U盘启动机器.
```
* 用户名: root  密码: xiange
* gpkg -I | sort | nl 可以看到所有已安装的软件包.
* gpkg -l 软件包名 可以看到包内文件.
* gpkg -c 软件包名 可以校验已安装的软件包.
* gpkg -s 软件包名 可以搜索查看软件包信息.
* gpkg -i 软件包名 可以安装软件包.
* gpkg -ip 软件包名 可以编译软件包但不安装.
* gpkg -ib 二进制包路径 可以安装二进制包.
* gpkg --sync 可以同步编译脚本库.
* gpkg -s 可以看编译脚本库中所有文件.
* gpkg edit 软件包名可以编辑编译脚本.
```**
